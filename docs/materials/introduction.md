# 企业介绍

四之堂始创于1914年（民国三年），创办人是海宁望族马家的马溯秋。堂名定为“四之”，寓意为“天护之、地养之、你病之、我医之”。四之堂药店悉尊古方，精选各省地道药材、饮片，虞修各类丸散膏丹，灶煎鹿、龟诸 胶而在当地享有相当的声誉。
民国初年，疫湿邪肆虐，百姓深受风湿、类风湿、痛风、历节病诸害之苦。海宁马桥老街的四之堂药店综合民间验方，创制祛风活血浸洗方，煎汤浸洗，熏蒸，手足患处关节10~20分钟，每日早晚一次，外治手足关节肿痛痹症，颇见效验，无毒副作用，深受民间好评。解放后，“四之堂”店主马介眉响应政府号召，汇集周边几家小药店，于1957年完成公私合营，成立马桥国新药合作商店，归属于马桥供销社。改革开放以来，公司升级改制，增资扩股，成为药品零售连锁经营企业，并于2013年5月恢复使用“四之堂”字号，成立嘉兴市四之堂医药连锁有限责任公司。经过近几年的发展，公司在海宁开设了10多家药店，4家中西医诊所。2019年经嘉兴市商务局认定为老字号企业。如今政通人和，百业兴旺，为发扬老店优良传统，重朔四之堂之美誉，集资数拾万元人民币，在发源地马桥桐木商业街创四之堂医馆，承先辈之仁德，邀请名老中医坐堂诊脉，服务百姓健康。
